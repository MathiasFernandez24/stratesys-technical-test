Proyecto Stratesys
Este es un proyecto desarrollado como parte de una prueba técnica para Stratesys. El proyecto es una aplicación móvil desarrollada en React Native que permite 
gestionar una lista de clientes.

Tecnologías Utilizadas:
-React Native: Framework de desarrollo de aplicaciones móviles multiplataforma.
-Expo: Plataforma y conjunto de herramientas para el desarrollo de aplicaciones React Native.
-React Navigation: Librería para la navegación en aplicaciones React Native.
-Formik: Librería para la gestión de formularios en React Native.
-Yup: Librería para la validación de esquemas en JavaScript.
-Context API: API de React para compartir datos entre componentes de forma eficiente.
-Fetch API: API nativa de JavaScript para realizar solicitudes HTTP.


Pasos a seguir para ejecutar el proyecto:

1- Clona este repositorio en tu máquina local:
    git clone https://github.com/MathiasFernandez24/stratesys-technical-test

2- Ingresa a la raiz del proyecto:
    cd stratesys-technical-test

3- Instala las dependencias en la raiz del proyecto:
    npm i
    ó
    yarn install

4- Ejecuta la aplicacion:
    npx expo start

5- Escanea el codigo QR con la app Expo go desde tu dispositivo o presiona la letra "a" si tienes un emulador de android corriendo o la letra "i" si tienes un emulador de Ios corriendo
